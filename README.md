# 🚗 Auto.KG - Mobile Marketplace

Мобильное приложение для покупки и продажи автомобилей в Кыргызстане.

## 🚀 Быстрый старт

### Backend:
```bash
cd backend
npm install
npm start
```

### Frontend:
```bash
cd AutoKG_Expo
npm install
npm start
```

## ✨ Основные функции

- ✅ Авторизация и регистрация
- ✅ Создание и редактирование объявлений
- ✅ Загрузка фото
- ✅ Real-time чаты (WebSocket)
- ✅ Система рейтингов и отзывов
- ✅ Поиск и фильтрация
- ✅ Избранное
- ✅ 3D-просмотр автомобилей

## 🛠 Технологии

- React Native + Expo
- Node.js + Express
- PostgreSQL
- Socket.IO
- JWT Authentication

## 📱 Платформы

- iOS
- Android
- Web (частично)
